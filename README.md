## 📌 Project Overview

**Simple Calculator** is a lightweight and interactive web application developed using frontend technologies.

The application allows users to perform basic mathematical calculations through a clean and responsive user interface.

This project demonstrates the implementation of:

- HTML page structure
- CSS styling and responsive design
- JavaScript logic and DOM manipulation
- User interaction handling

---

# ✨ Features

## 🔢 Calculator Operations

✔ Addition (+)  
✔ Subtraction (-)  
✔ Multiplication (*)  
✔ Division (/)  
✔ Percentage (%)  
✔ Decimal calculations  


## 🎨 User Interface

- Modern calculator design
- Responsive layout
- User-friendly buttons
- Real-time display updates
- Clean and simple interface


## ⌨️ User Input

Supports:

- Mouse click operations
- Number input
- Operator selection
- Clear/reset functionality
- Backspace/delete operations

# 🚀 How to Run the Project

## Step 1: Clone Repository

```bash
git clone <repository-url>
```

## Step 2: Navigate to Project Folder

```bash
cd Simple_Calculator
```

## Step 3: Run Application

Open:

```
index.html
```

in any modern web browser.

No installation or additional dependencies are required.
